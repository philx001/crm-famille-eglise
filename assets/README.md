# Assets (logos des familles)

Placez ici le logo de chaque famille pour l’affichage dans la sidebar.

## Logo Les Déterminés

1. **Nom du fichier à utiliser** : `logo-determines.png`
2. **Emplacement** : ce fichier doit être dans ce dossier `assets/`, donc à la racine du projet :  
   `assets/logo-determines.png`
3. **Contenu** : l’image du logo (personnages colorés + texte « LES DÉTERMINÉS » / « À ÊTRE DES DISCIPLES DE JÉSUS-CHRIST »).
   - Vous pouvez enregistrer l’image fournie (celle utilisée en pièce jointe) sous le nom `logo-determines.png` dans ce dossier.
4. En local : si vous ouvrez l’app en `file://`, certains navigateurs bloquent les images dans des dossiers. Lancez un serveur local (ex. `python -m http.server 8080`) à la racine du projet pour que le logo s’affiche.

Si le fichier est absent ou inaccessible, seul le slogan s’affiche dans la sidebar (sans icône cassée).

D’autres familles pourront avoir leur propre fichier (ex. `logo-autre-famille.png`) une fois la configuration étendue dans le code.
